Release Notes: GraSPy 0.0.X¶

We're happy to announce the release of GraSPy 0.0.X! GraSPy is a Python package for understanding the properties of random graphs that arise from modern datasets, such as social networks and brain networks.

For more information, please visit our website and our tutorials.
